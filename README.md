# Яйцо и время

Простой таймер для варки яиц.

Key features:
- Разные виды яиц
- Выбор степени готовности
- Забавный индикатор прогресса в форме яйца
- Поддерживается работа в фоновом режиме
- Приложение пришлет уведомление, когда яйцо будет готово
